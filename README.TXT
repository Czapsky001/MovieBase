Project Description
A simple web application that fetches data from an external movie API and displays it on the page.
The project aims to allow users to browse movie information and utilize login and registration functionalities.

Features
Displaying a list of movies from the API.
Ability to search for movies(after searching, movie images are incompleted css).
Login and registration panel (functions currently inactive).
User interface responsive to different devices, but currently experiencing layout issues after entering a search query.
